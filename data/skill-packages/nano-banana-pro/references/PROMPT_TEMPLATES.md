# Nano Banana Pro — Prompt Templates

Quick-copy templates for common image generation tasks.

---

## Product Photography

```
[PRODUCT NAME] product photography, [SPECIFIC ANGLE], studio lighting setup, 
professional white background, [ASPECT RATIO], [SIZE],
high-end commercial photography style, sharp focus on product details,
true-to-life colors, minimal shadows, [OPTIONAL: with human model holding/wearing product]
```

**Examples:**
```
Minimalist sneaker product shot, 45-degree angle, soft studio lighting, 
white background, 4:5 aspect, 2K, premium footwear photography

Coffee mug on marble surface, overhead view, natural window lighting, 
warm tones, 1:1 aspect, 2K, lifestyle product photography
```

## Brand Marketing Banner

headline "[EXACT TEXT]" in [FONT_STYLE], [SUBJECT],

## Style Transfer

```
Transform this image to match the [STYLE REFERENCE] style:
[SPECIFIC CHANGES], while preserving [ELEMENTS TO KEEP],
[ASPECT RATIO], [SIZE]
```

**Example:**
```
Transform this product photo to match the watercolor style reference,
keeping the exact product shape and branding visible,
apply loose brushstroke texture and soft color palette,
preserve lighting direction, 4:5, 2K
```

## Image Editing

```
Edit this image: [SPECIFIC CHANGE 1], [SPECIFIC CHANGE 2],
[AREA TO FOCUS ON], preserve [ELEMENTS TO KEEP IDENTICAL],
[ASPECT RATIO], [SIZE]
```

**Example:**
```
Edit this portrait: change background to sunset outdoor scene,
enhance eyes with more detail, remove stray hair strands,
preserve exact facial features and pose, 3:4, 2K
```
