## 1. Clarify visual job
Action: Define the exact deliverable, platform, and business goal
Verify: The prompt knows what is being generated and why

## 2. Lock brand identity
Action: Pull colors, typography, template rules, and logos
Verify: The prompt cannot drift into a generic look

## 3. Rank references
Action: Separate strict references from inspiration references
Verify: Reference hierarchy is explicit

## 4. Write master prompt
Action: Specify subject, composition, lighting, styling, and platform fit
Verify: Prompt is precise enough for repeatable rendering

## 5. Add guardrails
Action: State forbidden treatments and failure modes
Verify: Negative prompt closes the main risks

## 6. Prepare variations
Action: Create controlled brand-safe alternatives
Verify: Variations differ without breaking identity
