# Brand Template Enforcement — Instructions

## Brand Template Enforcement

1. **Identify fixed zones** — Determine what cannot move: logo lockups, title blocks, CTA zones, margins, safe areas.
   - ✓ Done when: Structural constraints are explicit.

2. **Lock the palette and type system** — Keep visual identity within the approved brand kit.
   - ✓ Done when: No off-brand substitutions slip in.

3. **Check readability and hierarchy** — Ensure text overlays are legible and visually prioritized correctly.
   - ✓ Done when: The design remains usable on its intended platform.

4. **Review platform fit** — Confirm the output works at the intended aspect ratio and safe zone.
   - ✓ Done when: The asset is platform-ready.

5. **Reject drift** — Flag any elements that feel like a generic AI aesthetic rather than the saved template system.
   - ✓ Done when: The brand identity remains intact.
