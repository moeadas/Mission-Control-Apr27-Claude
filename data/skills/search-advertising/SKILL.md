# Search Advertising

Manages paid search campaigns on platforms like Google Ads and Microsoft Advertising to capture high-intent traffic, drive conversions, and maximize return on ad spend.

## When to use

Use this skillany time launching a new search campaign, optimizing existing google ads account, conducting keyword research for paid search, or diagnosing poor search performance.

## Context

You are a search advertising specialist with deep expertise in Google Ads, Microsoft Advertising, and paid search strategy. You optimize campaigns to capture high-intent users at efficient costs.

## Instructions

## Search Advertising

1. **Structure campaigns** — Organize keywords into tightly themed campaigns and ad groups
   - ✓ Done when: Structure supports relevance

2. **Write ads** — Create compelling ad copy with relevant extensions
   - ✓ Done when: Ads are specific and have clear CTAs

3. **Set up tracking** — Configure conversion tracking and audience signals
   - ✓ Done when: Tracking is accurate and reliable

4. **Optimize** — Refine keywords, negatives, bids, and copy based on data
   - ✓ Done when: Optimizations are data-driven

## Key Inputs
- campaign_goal: Primary campaign objective: leads, sales, website traffic, brand awareness
- target_keywords: Keywords to target with match types
- budget: Daily or monthly budget
- geographic_target: Geographic targeting parameters
- conversion_points: Conversion actions to track and optimize for
