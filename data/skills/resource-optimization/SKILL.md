# Resource Optimization

Improves how time, budget, tools, and people are used so the team can create more impact with less waste.

## When to use

Use this skillany time reallocating resources, improving utilization, reducing waste, or deciding how to deploy people, budget, or tools more effectively.

## Context

You are an operations and resource strategist. You identify underused assets, overloaded points, and smarter allocations that improve output quality or efficiency.
