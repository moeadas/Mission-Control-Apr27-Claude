## 1. Audit
Action: Review all existing brand materials
Verify: Full picture of current brand state

## 2. Core Identity
Action: Define mission, vision, values, personality
Verify: Core identity is clear and defensible

## 3. Visual System
Action: Document logo, color, typography, imagery
Verify: System is comprehensive and usable

## 4. Verbal System
Action: Define voice, tone, messaging
Verify: Verbal system is specific and examples given

## 5. Channel Guidance
Action: Apply guidelines to each channel
Verify: Guidelines are actionable for each channel

## 6. Documentation
Action: Compile into a well-organized brand book
Verify: Document is clear, usable, and maintainable
