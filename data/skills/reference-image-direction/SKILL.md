# Reference Image Direction

Translates uploaded reference images, templates, and moodboards into precise production guidance so AI image generation follows the right visual cues without copying blindly or drifting off-brand.

## When to use

Use this skill any time a client provides reference images, moodboards, or templates that must guide a generated artwork, campaign visual, or branded concept.

## Context

You are a visual analyst who turns reference material into exact production guidance. You know how to distinguish structural template rules from softer inspiration cues.

## Instructions

## Reference Image Direction

1. **Catalog references** — List every uploaded logo, template, moodboard, and inspiration image.
   - ✓ Done when: Inputs are separated by purpose.

2. **Extract strict rules** — Identify the elements that must be preserved exactly.
   - ✓ Done when: The non-negotiables are explicit.

3. **Extract flexible cues** — Identify mood, texture, lighting, framing, and energy cues that can guide the output more loosely.
   - ✓ Done when: Inspiration is useful without becoming copying.

4. **Resolve conflicts** — If references contradict each other, decide which one wins and why.
   - ✓ Done when: Production has one coherent direction.

5. **Write execution notes** — Convert the reference analysis into actionable instructions for prompting and review.
   - ✓ Done when: The next agent can generate or evaluate the artwork confidently.
